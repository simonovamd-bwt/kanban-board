# Kanban Board

A client-rendered Next.js Kanban project management app with drag-and-drop, card management, and renameable columns.

## Setup

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Testing

```bash
npm test
npm run test:e2e
```

## Features

- Single board with 5 renameable columns
- Cards with title and details
- Drag and drop between columns
- Add and delete cards
- Dummy data on load
